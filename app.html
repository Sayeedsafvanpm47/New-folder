I Aid : Assistant App For The Visually Impared

Project information

  • Category: Android Application Development
  • Purpose: Final year project
  • Project date: 05/02/2023
  • Project name/strong>: I Aid : App for the Visually Impared Citizen

This application targets an audience who has limitations regarding visual capabilities and thus will ease the use of certain vital functions in mobile by operating using voice commands. I used android studio as the IDE where JAVA,Kotlin,Python,Machine learning etc was used as back-end and mainly XML for front-end development process. The algorithms like YOLO and TensorFlow lite were used for implementing the features like Object detection, Color detection, Gesture identification and Currency detection respectively. Java Speech to text (STT) and Text to speech (TTS) audio gateway has been used to configure the UI in the minds of the potential users so they can visualise the UI in their minds and operateÂ accordingly.
